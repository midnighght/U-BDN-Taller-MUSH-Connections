# MUSH

Una red social moderna desarrollada con React + TypeScript + Tailwind CSS y Nest.js + MongoDB + Neo4j para el curso de Taller de Base de Datos NoSQL.

##  Características

-  **Sistema de autenticación** con validaciones
-  **Interfaz responsive** que funciona en todos los dispositivos  
-  **Diseño moderno** con Tailwind CSS
-  **Desarrollo rápido** con Vite
-  **Fácil de extender** con arquitectura modular

##  Tecnologías Utilizadas

### Frontend
- **React 18** - Biblioteca de interfaz de usuario
- **TypeScript** - JavaScript tipado
- **Tailwind CSS** - Framework de CSS utility-first
- **Vite** - Herramienta de build y desarrollo rápido

### Backend (Próximamente)
- **NestJS** - Framework de Node.js
- **MongoDB** - Base de datos documental
- **Neo4j** - Base de datos de grafos
- **JWT** - Autenticación por tokens
